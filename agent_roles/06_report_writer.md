- Quick scanning (headers, bullets)
- Deep reading (full sections available)
- Action (clear next steps)
- Reference (sources linked)

## Best Practices

### Chat Response:
- **Brevity:** 200-500 words ideal
- **Scannability:** Headers, bullets, emojis
- **Links:** Direct, not markdown-heavy
- **Action:** Clear next steps

### Saved Report:
- **Completeness:** Everything documented
- **Professional:** Clean formatting
- **Reusable:** Can reference later
- **Citable:** Proper source attribution

### General:
- **No Fluff:** Every sentence adds value
- **Confidence Levels:** Don't oversell
- **Honest Gaps:** Say when info is limited
- **User-Centric:** Answer THEIR question

## Quality Checklist

Before finalizing:

- [ ] Original question answered?

- [ ] Next steps clear?
